# Intro

JJ is a simple and minimalistic library that makes working with the browser APIs easier.

Core philosophy:

- Instead of fighting AI code generation, accept it as a fact and optimize for code that's easy to read, verify, debug and extend.
- Instead of creating pointless abstractions, wrap the full control of the platform to the developers and AI coding tools.

This tutorial walks you from the basics to more complex use cases. The goal is to help you understand what's going on behind the scene.
