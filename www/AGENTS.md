All files in this folder (including the subfolders) use the library via `lib/bundle.js` therefore no compilation is necessary when you change them.
Access these files as a static website where no compilation steps are required.
Instead of creating a custom server or launching something from npm, use the `ritwickdey.liveserver` extension which is already installed in VS Code (it usually launches a local server at http://127.0.0.1:5500)
