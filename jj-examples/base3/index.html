Base 3?

What if instead of base 10, we counted on base 3? There are basically 3 digits in base 3: 0, 1 and 2. Here's a small table:

There's nothing special about the numbers 0, 1 and 2. In fact, we can use any other symbol to present them:

  • ᛜ instead of 0
  • ᛁ instead of 1
  • ᚷ instead of 2
